The Pipeline

How does it work?

  • Select

    Users can first select a data set that follows their criteria from OONI's database. Users have the option to select data by country code, date, asn, type of test, and url. Here are a couple examples: All Russian web connectivity measurements from January 2016 that test the URL roskomsvoboda.org; or all Chinese web connectivity measurements from March 2017.

  • Store

    Users can then store the selected data into the Pipeline's database. The storing process involves downloading the selected data set from OONI and organizing it into multiple tables in our database. Learn more about how the data is organized and what tables are available here.

  • Download

    Once the data set is stored, users have the option to extract and download the data/tables that they need!
